📊 Sales Analytics Dashboard – Northwind Traders

This project presents a comprehensive Sales Analytics Dashboard built using the Northwind dataset. It provides actionable insights into sales performance, customer behavior, employee contribution, and supplier distribution through interactive visualizations.

🚀 Project Overview

The goal of this project is to analyze business performance and uncover key insights that support data-driven decision-making. The dashboard covers multiple dimensions including sales trends, product performance, customer segmentation, and regional distribution.

📌 Key Features
Sales Performance Analysis
Total sales, orders, customers, and products overview
Time-based sales trends (monthly/yearly)
Product & Category Insights
Top-performing products
Sales distribution across categories
Customer Analytics
Customer distribution across countries
Top customers by revenue
Segmentation into High, Medium, and Low value customers
Employee Performance
Sales contribution by each employee
Year-over-year performance trends
Employee tenure analysis
Supplier Insights
Sales contribution by suppliers
Regional supplier distribution
Product supply variation
📈 Key Insights
Sales show an overall upward trend with peak performance in specific periods.
A small number of products and customers contribute significantly to total revenue.
The USA and European countries dominate customer and sales distribution.
Employee performance varies, with top contributors driving major sales.
Supplier contribution is uneven, with dependency on a few key suppliers and regions.
🛠️ Tools & Technologies
SQL – Data extraction and transformation
Power BI / Tableau – Data visualization and dashboard creation
Excel – Data preprocessing and validation
📂 Dataset
Northwind Sample Database
Includes tables such as Customers, Orders, Products, Suppliers, and Employees
